Personal dotfiles, [managed with GNU Stow] [1]. Inspiration (and, in many
cases, actual code):

* [Mathias Bynens's dotfiles] [2]
* [Seth House's `.pythonrc`] [3]

To install the dotfiles, install [GNU stow] and run:

`stow bash curl python screen system`
